Problem Statement

  • Educational institutions struggle to monitor student performance continuously.
  • Weak students are usually identified only after final examinations.
  • Manual analysis of academic records is inefficient.
  • Machine learning techniques can identify patterns in academic data.
  • Goal: Predict student performance early to support academic improvement.

Abstract

This project presents an AI based system for predicting student academic performance using machine learning techniques. Student academic details such as attendance, assignments, midterm marks and final exam scores are used to evaluate performance. The system calculates the average marks and determines whether the student will pass or fail. This helps teachers identify students who require additional academic support at an early stage.

Methodology

  • Collect student academic information such as attendance percentage, assignment marks, midterm marks, and final exam marks.
  • Student details are entered by the user at runtime using Python input statements.
  • The program calculates the average marks using mathematical expressions.
  • Conditional statements check whether the average marks are greater than or equal to 50.
  • Based on the condition, the system determines whether the student result is Pass or Fail.
  • Finally, the program displays the complete student report including student ID, name, attendance, average marks, and final result.

Modules with Description

1. Input Module
Collects student details such as ID, name, attendance and marks.
2. Calculation Module
Computes the average marks from assignment, midterm and final exam.
3. Decision Module
Uses conditional statements to determine whether the student passes or fails.
4. Output Module
Displays the complete student report including result.

Module Execution using Python



student_id = int(input("Enter Student ID: "))
name = input("Enter Student Name: ")

attendance = float(input("Enter Attendance Percentage: "))
assignment = float(input("Enter Assignment Marks: "))
midterm = float(input("Enter Midterm Marks: "))
final_exam = float(input("Enter Final Exam Marks: "))

average = (assignment + midterm + final_exam) / 3

if average >= 50:
    result = "Pass"
else:
    result = "Fail"

print("\n--- Student Report ---")
print("Student ID:", student_id)
print("Name:", name)
print("Attendance:", attendance)
print("Average Marks:", round(average, 2))
print("Final Result:", result)
